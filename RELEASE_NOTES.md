## Конвейер копирования (FolderSync) — сборка для Windows

Готовый `.exe`, не требует установки Python. Позволяет синхронизировать
**несколько папок и файлов одновременно** (каждое правило копируется в
своём собственном потоке), с индивидуальными настройками для каждого
правила и веб-интерфейсом, защищённым ключом безопасности.

### Что нового в этой версии
- Параллельная синхронизация: все включённые правила работают одновременно,
  а не по очереди.
- Индивидуальные настройки на правило: свой интервал проверки стабильности
  и своё поведение «пропускать неизменные файлы» — либо использовать общие.
- Ключ безопасности для доступа к веб-интерфейсу: генерируется автоматически
  при первом запуске, нужен для входа в браузере и для запросов к API
  (заголовок `X-API-Key`). Защита от подбора ключа (блокировка после
  нескольких неверных попыток).

### Как запустить
1. Скачайте `FolderSync.exe` из раздела **Assets** ниже.
2. Запустите файл. Откроется консольное окно — в нём будет выведен
   **ключ безопасности**, он же будет автоматически сохранён в файл
   `secret.json` рядом с exe.
3. Браузер откроется автоматически на `http://127.0.0.1:5000`. Если этого
   не произошло — откройте адрес вручную.
4. При первом входе вставьте ключ безопасности из консоли.
5. Добавьте одно или несколько правил копирования («+ Добавить правило»)
   и нажмите «Запустить».

### Если Windows/SmartScreen предупреждает о неизвестном приложении
Это стандартное поведение для исполняемых файлов без цифровой подписи.
Нажмите **«Подробнее» → «Выполнить в любом случае»**, если доверяете
источнику (официальный релиз этого репозитория на GitHub).

### Файлы, которые создаёт программа рядом с exe
- `config.json` — список правил копирования и настройки;
- `secret.json` — ключ безопасности и секрет сессии (никому не передавайте).

Оба файла можно удалить, чтобы сбросить настройки/ключ — они будут
пересозданы при следующем запуске.

### Контрольная сумма
После публикации релиза рекомендуется свериться с SHA256 из вкладки
**Assets**, если она указана, перед запуском exe на важном компьютере.
